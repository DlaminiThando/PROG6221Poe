# Cybersecurity Awareness Bot



\## Project Description

This project is a console-based cybersecurity awareness chatbot developed as part of the PROG6221 Programming assignment. The bot provides educational information about cybersecurity topics including password safety, phishing attacks, and safe browsing practices.



\## Features Implemented



\### 1. Voice Greeting

\- Plays a recorded voice greeting when the application starts

\- Audio file: `greeting.wav`



\### 2. ASCII Art Logo

\- Displays a custom ASCII art logo for the Cybersecurity Awareness Bot

\- Creates visual appeal and branding



\### 3. Text-Based Greeting and User Interaction

\- Asks user for their name

\- Personalizes responses using the user's name



\### 4. Basic Response System

\- Responds to questions about cybersecurity

\- Topics covered:

&#x20; - Password safety

&#x20; - Phishing awareness

&#x20; - Safe browsing

&#x20; - General cybersecurity tips



\### 5. Input Validation

\- Handles empty entries

\- Provides helpful error messages

\- Responds with "I didn't quite understand that. Could you rephrase?" for invalid inputs



\### 6. Enhanced Console UI

\- Colored text for different message types

\- Borders and dividers for visual structure

\- Typing effect for realistic conversation feel



\### 7. GitHub Version Control

\- Minimum of 6 meaningful commits

\- Descriptive commit messages

\- GitHub Actions CI workflow



\## How to Run

1\. Ensure you have .NET 9.0 installed

2\. Clone this repository

3\. Open the project in Visual Studio

4\. Press F5 to run the program



\## Requirements

\- .NET 9.0 SDK

\- Windows OS (for voice playback)
